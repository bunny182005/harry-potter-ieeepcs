üìú About The Project
This platform is the central hub for the "PotterNova" event. It was built to manage the entire user journey, from registration to the final round of the competition. The site facilitates user authentication, team creation, and real-time scoring across five distinct rounds, all wrapped in a magical Harry Potter theme.

‚ú® Core Features
User Authentication: Secure Sign Up and Login system for all participants using Firebase.

Team Formation: Allows users to create and join teams to compete together.

Live Leaderboard: A dynamic leaderboard that updates in real-time as teams score points.

Multi-Round Gameplay: Hosts 5 unique rounds, with challenges and quizzes played directly on the site.

Event Information: Provides all the necessary details and rules for each round of the PotterNova event.

Responsive Design: Fully functional and accessible on both desktop and mobile devices.

üöÄ How To Run Locally
To get a local copy up and running, follow these simple steps.

Clone the repository:

git clone [https://github.com/bunny182005/harry-potter-ieeepcs.git](https://github.com/bunny182005/harry-potter-ieeepcs.git)

Navigate to the project directory:

cd harry-potter-ieeepcs

Install NPM packages:

npm install

Run the development server:

npm run dev

This will start the application on a local server, typically http://localhost:5173.
